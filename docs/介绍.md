# ClipVault 功能介绍

> **ClipVault** 是一款 Chrome 扩展（Manifest V3），用于记录剪贴板历史、管理常用文本片段，并在已打开的标签页中快速搜索内容。数据默认保存在浏览器本地，注重隐私与可控性。

---

## 产品定位

Chrome 没有内置「剪贴板历史」，复制后一旦覆盖就难以找回。ClipVault 在后台自动记录复制内容（可配置关闭或按站点排除），并提供搜索、收藏、片段库与可选 AI 增强，适合写作、开发、客服、研究等需要频繁复制粘贴的场景。

---

## 核心功能

### 1. 剪贴板历史

- **自动捕获**：在网页上复制文本后自动写入历史（可在设置中关闭「自动监听」）。
- **智能类型识别**：自动识别链接、邮箱、电话、代码与普通文本，并提供对应快捷操作（如打开链接、撰写邮件）。
- **搜索与排序**：在历史中按关键词搜索；支持按时间、字数、来源排序。
- **收藏与批量**：单条收藏；批量选择后收藏或删除。
- **展开长文本**：超长内容预览截断，可展开查看全文。
- **来源信息**：记录复制时的页面标题与来源，便于回溯。
- **快捷键粘贴**：在 Popup / 侧边栏中，按 `1`–`9` 快速复制对应条目（列表前 9 项）。

**默认策略（可在设置中修改）：**

| 配置项 | 默认值 |
|--------|--------|
| 历史上限 | 500 条 |
| 单条最大长度 | 5000 字符 |
| 自动清理 | 30 天 |
| 重复过滤间隔 | 60 秒（相同内容短时间内不重复记录） |

---

### 2. 文本片段（Snippets）

- 保存常用文本（签名、模板、代码片段等）。
- **分组与标签**：按分组筛选；支持「未分组」。
- **新建 / 编辑 / 删除**：在片段标签页内联编辑。
- **搜索**：按标题或内容过滤片段。
- **一键复制**：每条片段可快速复制到剪贴板。

---

### 3. 跨标签页搜索

在「搜全站」标签页中，对**当前浏览器窗口已打开的所有标签页**进行文本搜索：

- 支持**区分大小写**、**正则模式**（可在设置或 Popup 工具栏中配置）。
- 展示每个标签页的匹配数量与摘要片段。
- 点击结果可**跳转到对应标签页**并在页面内高亮匹配位置。

---

### 4. 网页悬浮球

在任意网页右下角显示 ClipVault 悬浮球（可在设置中关闭）：

| 操作 | 说明 |
|------|------|
| 点击悬浮球 | 打开剪贴板历史面板，点击条目即可复制 |
| 悬停后点击 × | 打开悬浮球设置（缩小、隐藏方式等） |
| 拖动 | 可拖动悬浮球到合适位置 |

**隐藏方式：**

- 本次访问关闭（关闭标签页或重新打开页面前不再显示）
- 永久禁用（在扩展设置页重新开启）
- 仅当前网站禁用（可在设置页管理已禁用站点列表）

---

### 5. 多种界面入口

| 入口 | 说明 |
|------|------|
| **工具栏 Popup** | 点击扩展图标，固定布局：顶栏 + 搜索/标签 + 可滚动列表 + 底栏 |
| **侧边栏（Side Panel）** | 更大空间浏览历史与片段；需 Chrome 116+ |
| **设置页** | 完整配置项、数据导入导出、错误日志 |

**推荐快捷键：**

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+V`（Mac：`⌘+Shift+V`） | 打开 Popup |
| `Ctrl+Shift+S`（Mac：`⌘+Shift+S`） | 切换侧边栏（需从 Popup 内打开侧栏时受 Chrome 用户手势限制） |

---

### 6. 外观与语言

**外观模式**（设置 → 外观模式）：

- **跟随浏览器**：与 Chrome 浅色/深色设置一致（亦会随系统外观变化）
- **浅色** / **深色**：固定配色

适用于 Popup、侧边栏、设置页及网页悬浮球。

**界面语言**：

- 跟随浏览器 / English / 中文

---

### 7. AI 功能（可选，默认关闭）

在设置中启用 AI 后，可对剪贴板条目进行增强处理。**数据通过您选择的通道处理，不会经过 ClipVault 自有服务器。**

#### 提供方

| 通道 | 说明 |
|------|------|
| **API 提供商（推荐）** | 使用您自己的 API Key，支持 OpenAI 兼容接口 |
| **Chrome 内置 AI** | 本地模型；扩展环境中能力受限，以设置页提示为准 |

**预设 API 提供商：** OpenAI、DeepSeek、智谱 GLM、通义千问、Moonshot、Ollama（本地）、自定义接口。

#### 功能开关

- **自动标签**：为新剪贴板条目生成标签
- **智能（语义）搜索**：在历史中按语义扩展检索
- **翻译**：将条目翻译为指定语言
- **改写**：润色或改写文本
- **摘要**：生成内容摘要（历史项操作）

历史列表中可对单条记录使用 **摘要 / 翻译 / 改写** 等操作（视开关而定）。支持**测试连接**与**每分钟调用上限**配置。

> **隐私提示：** API 密钥仅保存在本地 `chrome.storage`；调用第三方 API 时，相关内容会发送至对应服务商，请自行查阅其隐私政策。

## 隐私与安全

- **本地存储**：历史、片段、设置均保存在浏览器 `chrome.storage.local`，不上传到 ClipVault 服务器。
- **排除网站**：可为银行、内网等敏感域名配置「排除网站」，不在这些站点记录复制内容。
- **用户控制**：可关闭自动监听、关闭悬浮球、关闭 AI；可一键清空全部数据。
- **权限说明**：扩展需要剪贴板读取、访问网页（悬浮球与跨标签搜索）等权限；仅用于上述功能。

## 系统要求

| 项目 | 要求 |
|------|------|
| 浏览器 | Google Chrome **116+**（侧边栏功能） |
| 扩展标准 | Manifest V3 |
| 最低版本 | `manifest.json` 中 `minimum_chrome_version: "116"` |

低于 116 的 Chrome 可使用 Popup 与大部分功能，但侧边栏入口不可用。

## 典型使用场景

1. **写作 / 翻译**：复制多段参考资料 → 在历史中搜索 → 用 AI 翻译或改写 → 复制结果。
2. **开发**：复制报错、API 地址、代码块 → 类型自动识别为代码/链接 → 跨标签搜索相关文档页。
3. **客服 / 运营**：常用回复存入片段 → 分组管理 → Popup 中 `1`–`9` 快速粘贴。
4. **研究**：多标签查阅资料 → 「搜全站」定位哪一页包含关键词 → 跳转并高亮。
